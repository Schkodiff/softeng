Сценарий учёта сотрудников:
	Предусловие:
	1. Находится в главном меню

	Просмотр списка картин:
		1. В панели навигации нажимает на кнопку "Списки" -> "Сотрудники"
		2. Открывается окно со списком сотрудников.

		Альтернатив:
			1. Нажимает CTRL+L для появления меню списка
		Исключения:
			1. Если база данных не доступна в данный момент, выводит окно с сообщением "База данных не доступна".

	Предусловие:
	1. Находится в списке сотрудников

	Редактирование:
		1. Админ выбирает одного сотрудников
		2. Нажимает на кнопку редактирования сотрудников
		3. Появляется окно редактирования сотрудников
		4. В окне редактирования можно изменить ФИО, дату рождения, должность, номер телефона
		5. По завершению изменения данных, админ нажимает нажимает на кнопку Сохранить редактирование
		6. Программа сохраняет изменения и закрывает окно редактирования
		7. Админ попадает в обновленный список сотрудников

		Альтернатив:
			2. Нажимает CTRL+R для открытия окна редактирования
			5. Нажимает CTRL+S для сохранения.
			5. Нажимает Отмена
				5.1. Программа закрывает окно редактирования
				5.2. Админ попадает в неизмененный список сотрудников
		
		Исключения:
			1. Если пропустил обязательное поля ввода, то выводит на экран сообщение "Пропустили поле";

------------------------------------------------------------------------------------------------------------------------------------------------
	Добавление:
		1. Нажимает на кнопку добавления сотрудников
		2. Появляется окно добавления сотрудников
		3. В окне добавления нужно вписать ФИО, дату рождения, должность, номер телефона
		4. По завершению ввода данных, админ нажимает нажимает на кнопку Добавить
		5. Программа сохраняет добавление и закрывает окно добавления
		6. Админ попадает в обновленный список сотрудников
		
		Альтернатив:
			1. Нажимает CTRL+A для открытия окна добавления
			4. Нажимает CTRL+S для сохранения.
			5. Нажимает Отмена
				5.1. Программа закрывает окно добавления
				5.2. Админ попадает в неизмененный список сотрудников		

		Исключения:
			1. Если пропустил обязательное поля ввода, то выводит на экран сообщение "Пропустили поле";

------------------------------------------------------------------------------------------------------------------------------------------------
	Поиск:
		1. Нажимает на поле поиска в правом верхнем углу
		2. Вводит имя сотрудника (частично или полностью) или его номер
		3. Нажимает на кнопку поиска
		4. Программа осуществляет поиск по введенным критериям и показывает результаты.
		
		Альтернатив:
			1. Нажимает CTRL+F для поиска картин.
		
		Исключения: 
			1. Если название введено некорректно, то выводит на экран сообщение "Пусто";

------------------------------------------------------------------------------------------------------------------------------------------------
	Удаление:
		1. Админ выбирает одного или нескольких сотрудников
		2. Нажимает на кнопку удаления сотрудников
		3. Появляется окно подтверждения с кнопками "Ок" и "Отмена"
		4. При нажатии на кнопку подтверждения, программа удаляет выбранные объекты и возвращает админа обратно в список
		
		Альтернатив:	
			2. Нажимает на клавишу "Del"
			3. Нажимает на "Enter" для подтверждения или "Esc" для отмены
			4. При нажатии "Отмена", программа возвращает админа обратно в список, не удаляя объекты

		
Сценарий учета картин:
	Предусловие:
	1. Находится в главном меню

	Просмотр списка картин:
	1. В панели навигации нажимает на кнопку "Списки" -> "Картины"
	2. Открывается окно со списком картин.
		Альтернатив:
			1.Нажимает CTRL+L для появления меню списка
		Исключения:
			1. Если база данных не доступна в данный момент, выводит окно с сообщением "База данных не доступна".

	Предусловие:
	1. Находится в списке картин

	Редактирование:
		1. Админ выбирает картину
		2. Нажимает на кнопку редактирования картин и появляется окно редактирования картин;
		3. В окне редактора картин сотрудник имеет возможность редактировать данные выбранной картины;
		4. По завершению редактирования, сотрудник нажимает на кнопку сохранения изменений, программа сохраняет введенные изменения и закрывает окно редактирования
		5. Сотрудник попадает в обновленный список картин.

		Альтернатив:
			1. Нажимает CTRL+R для открытия окна редактирования
			3. Нажимает CTRL+S для сохранения.
			4. Нажимает Отмена
				4.1. Программа закрывает окно редактирования
				4.2. Админ попадает в неизмененный список картин
		
		Исключения:
			1. Если пропустил обязательное поля ввода, то выводит на экран сообщение "Пропустили поле";	

------------------------------------------------------------------------------------------------------------------------------------------------
	Добавление:
		1. Нажимает на кнопку добавления картин и появляется окно добавления картин;
		2. В окне редактора картин сотрудник имеет возможность добавить данные новых картин;
		3. По завершению ввода данных картин, сотрудник нажимает на кнопку сохранения, программа сохраняет добавление и закрывает окно
		4. Сотрудник попадает в обновленный список картин.

		Альтернатив:
			1. Нажимает CTRL+A для открытия окна добавления
			3. Нажимает CTRL+S для сохранения.
			3. Нажимает Отмена
				3.1. Программа закрывает окно добавления
				3.2. Админ попадает в неизмененный список картин
		Исключения:
			1. Если пропустил обязательное поля ввода, то выводит на экран сообщение "Пропустили поле";

------------------------------------------------------------------------------------------------------------------------------------------------
	Отчет:
		1. Админ выбирает одного или нескольких картин
		2. Нажимает на кнопку составления отчета по картинам и появляется окно составления отчёта;
		3. В этом окне сотрудник имеет возможность выбрать между 3 режимами составления отчёта:
			1-ый режим составляет отчёт по количеству картин, находящихся на реставрации, а также какие именно картины;
			2-ой режим составляет отчёт по количеству картин, находящихся на экспозициях, а также какие именно картины и где проходят экспозиции;
			3-ий режим составляет отчёт по количеству картин, находящихся в хранилище, а также какие именно картины.
		4. По завершению составления отчета картин, сотрудник нажимает на кнопку сохранения, программа сохраняет отчет и закрывает окно
		5. Сотрудник попадает в обновленный список картин.
		
		Альтернатив:
			1. Нажимает CTRL+O для открытия окна составления отчета
			3. Нажимает Отмена
				3.1. Программа закрывает окно отчёта
				3.2. Админ попадает в список картин
			4. Нажимает Отмена
				4.1. Программа закрывает окно отчёта
				4.2. Админ попадает в список картин		
------------------------------------------------------------------------------------------------------------------------------------------------
	Поиск:
		1. В правом верхнем углу, нажимает на поле поиска, где можно осуществить поиск по названию картины (частично или полностью) или по его идентификатору, после нажимает на кнопку поиска;
		2. Или нажимает на кнопку фильтр (в правом верхнем углу), где можно искать по жанрам, годам, авторам и странам авторов и нажимает на кнопку поиска. 
		
		Альтернатив:
			1. Нажимает CTRL+F для поиска картин.
		
		Исключения: 
			1. Если название введено некорректно, то выводит на экран сообщение "Пусто";

		1. В правом верхнем углу, нажимает на поле поиска, где можно осуществить поиск по названию жанра (частично или полностью) или по его идентификатору, после нажимает на кнопку поиска;
		
		Альтернатив:
			1. Нажимает CTRL+F для поиска жанра.
		
		Исключения: 
			1. Если название введено некорректно, то выводит на экран сообщение "Пусто";
